## 📊 Project Overview

This project showcases an interactive Tableau dashboard developed using structured sales data. It highlights key business insights such as sales performance, product trends, and regional analysis through visually engaging charts and dashboards. The dashboard includes filters and interactive features for dynamic data exploration. This project demonstrates strong skills in data visualization, storytelling, and transforming raw data into actionable insights using Tableau.
